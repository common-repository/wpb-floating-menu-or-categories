== Description ==

WPB Floating Menu or Categories is a free WordPress plugin, it allows you to increase your site usability by adding sticky floating side menu or categories with icons.

It works with any WordPress theme. It has a color customization option to match with any design layout. 

Our plugin support Menu Icons plugin by ThemeIsle. You can add icons to your menu items and our floating menu will show the icons.

Our plugin has an option for showing floating categories with icons as well. You can add FontAwesome icons on your category items and our floating categories plugin will show the icons.

It has two shortcodes to show the Floating menu or Categories to any specific page. Or you can set it globally for the entire site.

### Plugin Features:

* Floating menu.
* Floating category for any custom taxonomy.
* Floating menu and categories icon.
* Two different skins.
* Color customizer to match with any theme.
* Right or left placement option.
* Any menu or any custom taxonomy can use us Floating menu or category.
* Option for enable or disable the Floating menu or category on phone.
* Material design, mobile responsive and modern clean design.
* Easy ShortCode system.

* All Modern Browser Support. 
* Translation & RTL Language ready. 
* Very Lightweight & many More
